# 🧠 NeuroLearn AI v2

NeuroLearn AI v2 is an AI-powered smart learning platform that helps students learn faster, practice better, and track their progress using gamification. It combines AI tutoring, quiz generation, document understanding, and AWS-based progress tracking into a single system.

---

## 🚀 Features

### 🤖 AI Learning Assistant
Ask any question and get structured explanations using LLM (Groq AI).

### 📝 Smart Quiz Generator
Generate quizzes instantly on any topic and test your knowledge.

### 📊 Progress Tracking System
Gamified learning system with:
- XP system
- Level progression
- Streak tracking
- Achievement badges

### 📂 Document Q&A
Upload PDF/DOCX files and ask questions directly from content.

### ☁️ AWS Integration
- User data stored in AWS DynamoDB
- Quiz history stored in cloud database
- Progress tracking per user
